# ⚔️ اردوگاه قهرمانان

پروژه‌ای ساده برای ثبت و نمایش قهرمانان با قدرت‌های مختلف.

## تکنولوژی‌ها
- HTML, CSS, JavaScript (Frontend)
- Node.js + Express (Backend)
- MySQL (Database)

## نحوه اجرا
1. دیتابیس MySQL را راه‌اندازی کنید.
2. فایل `server.js` را با `node server.js` اجرا کنید.
3. مرورگر را باز کنید و به `http://localhost:1100` بروید.
